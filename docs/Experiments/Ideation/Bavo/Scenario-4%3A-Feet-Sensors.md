# Feet Sensors

The ability to mirror the footsteps in the virtual world with visual and sound feedback. You would get another sound depending on the ground type and an imprint in the ground if it is for example in mud.

![foot-sensor-diagram.jpg](/.attachments/foot-sensor-diagram-e9f482c8-5176-4c36-b55d-d7dde36a4b49.jpg)

## Desired Outcome

Creating extra feedback to the user to heighten immersion.
